/// SYSTEM LOGIC: VARIABLE LANDING ///

1. The "Brain" (Randomization Engine)
Every 15 seconds, a JavaScript function randomly selects one of three flight scenarios: PERFECT, HIGH, or LOW.

2. Visual Synchronization (3 Layers)
Once a scenario is active, three systems update in unison:

  • Flight Path Physics: The plane's descent angle, flare, and nose-drop (derotation) are calculated mathematically.
  • PAPI Lights: The 4-light ground system changes colors based on the active class.
  • Flight Instruments (PFD): The Glideslope diamond moves physically on the scale.

3. Go-Around Logic (New)
Clicking TO/GA interrupts the CSS animation engine. It captures the aircraft's realtime X/Y coordinates via the DOM, injects them into CSS Variables, and triggers a new "Climb Out" animation profile with simulated spool-up physics and gear retraction.
